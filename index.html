Dashboard Rules

How the system ranks stories and marks clocks.

Move
Market clocks use each exchange city’s local timezone. On Saturday/Sunday in that market’s home timezone, the market is always shown closed. Overall news score = 45% source reliability + 55% story score. If live RSS fails, the dashboard falls back to a built-in independent macro watchlist so the news panel never goes empty.
